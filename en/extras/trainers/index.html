    Trainers#

    Trainers are essentially modmenus that add a lot of fun and useful options. They're also a better alternative to the very limited amount of cheatcodes the game offers. You may want one for multiplayer, but make sure your server allows them.

    ZMenuIV#

    Compatibility

    This trainer is not compatible with the Complete Edition. Downgrade if using the Complete Edition.

    The most powerful and full-fledged trainer for GTA IV. Also includes a Chaos mod for streamers and fun. And... Goldsrc physics...? Bhopping is fun, I guess.

    Installation
    • Get the latest Ultimate ASI Loader.
    • Go to Zolika1351's Zone.
    • Scroll to the bottom of the page and download the latest version.
    • Extract ZMenuIV_vx.zip into the game folder (excl. folders asi loader if not using xliveless, for developers, old version (has VR support), parachute support(all into game folder). Read the readme.txt for details on these folders).
    Addons

    Read the readme.txt for information on the addons.

    Usage
    • The default controls are:
      • F7 to toggle menu.
      • Num 8, Num 2, Num 4, Num 6 for navigation. Num 5 to accept, Num 0 to go back.
      • Right Ctrl+Num 1 to fix the car.
      • Right Ctrl+Num 2 to flip the car.
      • ] (hold) to speed the car up.
      • F6 for Airbreak(noclip).
      • Right Ctrl for Special God 1.
      • F1 for Special God 2.
      • 0 for Always God Mode.
      • [ for Never Wanted.
      • ; for Unlimited Ammo.
      • = for Collision.
      • - for Resurrect.
      • ' for Forward Through Door.
      • M for Ragdoll.

    You can add your own custom keybinds that can combine multiple options together. To do so, enable Custom Keybind Creator in Trainer Settings, highlight the desired option and press Left Ctrl+Enter to set the keybind up.

    Edit the [Keybinds] section in ZMenuIV.ini to change existing keybinds.

    Setting up Numpad-less keybinds

    Not everyone has a Numpad, I get it, 80% (and less) keyboards are cool and comfortable. Open ZMenuIV.ini and locate these lines:

    [Keybinds]
AirbreakUp=87
AirbreakDn=83
AirbreakForward=104
AirbreakBack=98
AirbreakLeft=100
AirbreakRight=102
MenuUp=104
MenuDn=98
MenuLeft=100
MenuRight=102
MenuEnter=101
MenuBack=96

    and change them to:

    [Keybinds]
AirbreakUp=69
AirbreakDn=81
AirbreakForward=87
AirbreakBack=83
AirbreakLeft=65
AirbreakRight=68
MenuUp=38
MenuDn=40
MenuLeft=37
MenuRight=39
MenuEnter=13
MenuBack=8
    • Which will be:
      • Arrow keys for navigation. Enter to accept, Backspace to go back.
      • WASD for horizontal airbreak control, Q and E for vertical.

    Enable Disable Controls in Menu in Trainer Settings to avoid the phone taking over.

    Liberty's Legacy#

    Compatibility

    This trainer is compatible with the Complete Edition, as well as patches 1.0.8.0 and 1.0.7.0.

    This trainer is not as powerful as ZMenuIV, but still offers a lot of useful features, especially if you're playing with the Complete Edition.

    Installation
    • Get the latest Ultimate ASI Loader and ScriptHook.
    • Go to the GTAForums page.
    • Download the latest version.
    • Extract Liberty's Legacy Trainer xxx.zip into the game folder.
    Usage
    • The default controls are:
      • F11 or RB+X/R1+Square to toggle menu.
      • Arrow keys or D-Pad for navigation. Enter or A/X to accept, Backspace or B/Circle to go back.

    Additional key bindings can be set by highlighting the desired option, pressing Left Ctrl+Enter and pressing the key you want to bind.

    Simple Native Trainer#

    Compatibility

    This trainer is compatible with all patches.

    This trainer is less robust than the other two, but.. some people would want it, I guess.

    Installation
    • Get the latest Ultimate ASI Loader and ScriptHook.
    • Go to the GTAForums page.
    • Download latest version (it's somewhere in the post).
    • Extract trainer.asi, trainer.ini, trainertbogt.asi, trainertbogt.ini, trainertlad.asi, trainertlad.ini from trainervxx.rar into the game folder.
    Usage

    F3 to open the menu, Num 8, Num 2, Num 4, Num 6 for navigation. Read the readme.doc or the GTAForums page for instructions and keybinds.
