    Introduction#

    Welcome to Gillian's Grand Theft Auto IV Modding Guide.

    This is a centralized, all-in-one guide. It includes steps for optimizing, tweaking, fixing, improving the game, experiencing multiplayer and more!

    This guide is not affiliated with Rockstar Games, Rockstar North, or Take-Two Interactive.     Prerequisites#

    • In order for this guide to be useful to you, please make sure that the following requirements are met:
      • A PC running Windows 10/11 or any up-to-date Linux distribution suited for gaming.
      • You must own the game on Steam or the Rockstar Games Launcher. Pirated versions are not supported. Retail versions with 1.0.8.0 are not supported, but should work as long as you get rid of the DRM.
      • Your PC must at least meet the minimum requirements of the game. For a pleasant experience, however, it should also meet DXVK's requirements(ignore Linux requirements such as Wine if you're running Windows; older GPUs may meet the 1.10.3 requirements instead).
      • The game should not be located in C:\Program Files or C:\Program Files (x86) to avoid permission issues. This is irrelevant for Steam users.
      • Make sure you have enough free space on your drive (the base game takes about 22GB, a backup will take just as much, plus count in all the modding you'd do - reserve at least 70GB counting the game too).
      • You need to have an archivator installed. I recommend 7-Zip.
      • Make sure you have the game's prerequisites installed(they're already taken care of by Steam and the Rockstar Games Launcher) - Visual C++ Redistributable for Visual Studio 2012 x86 and DirectX End-User Runtimes (June 2010).
