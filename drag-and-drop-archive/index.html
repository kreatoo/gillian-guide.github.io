    Drag-and-Drop Archive#

    If you don't want to manually mod your game, you can choose an archive depending on the version you want - be it 1.2.0.59 or 1.0.8.0(a downgrader is included if you're using the Steam or Rockstar Games Launcher version). You can compare the modlist between the versions below - 1.0.8.0 has more, keep that in mind. You'll mainly be lacking ZolikaPatch and Liberty Tweaks.

    Notes#

    1. 1.2.0.59 is the Complete Edition, also known as the latest Steam or Rockstar Games Launcher version. It removes support for multiplayer and Games for Windows - LIVE, while adding the Rockstar Games Launcher (and it's DRM) and the Social Club overlay (with it's achievements). The amount of supported mods is also much more limited in this version - most mods are built for 1.0.8.0 and 1.0.7.0.
    2. 1.0.8.0 is the latest retail patch, with ZolikaPatch and multiplayer support. However, this drag-and-drop archive does NOT include any support for Games for Windows - LIVE whatsoever. See the downgrading and multiplayer sections instead. This version is better for mod compatibility.

    I can't update the archive too frequently myself - check out the pages to see if the mods, especially the ones ontop, got any updates.

    Make sure latest drivers are installed.

    Installation#

    Download

    Download the archive and then simply extract the contents into your game folder. After installation, go through additional setup.

    Warning

    The archive must be installed on top of a clean, unmodded Steam Complete Edition installation.

    If you are using the Rockstar Games Launcher version, do not start the game from the launcher itself, use PlayGTAIV.exe instead - otherwise the game files will be replaced.

    Other installation methods are not supported.

    In addition, I will not support any additional modifications to the files other than the instructions already listed.

    Updating

    If you're updating after installing the archive instead, delete update and modloader first.

    If the game does not start

    Try to install vcredist_x86.exe that's in your game folder.

    Alternatively, your PC may not support DXVK - remove d3d9.dll from the game folder or try other versions from additional mods.

    My game is behaving strangely/my game is crashing randomly

    Disable mods one by one to see the culprit by editing modloader.ini in modloader or deleting mods in update. Let me know if you find the problem.

    Additional Mods#

    These mods are not included by default, but do not require any additional steps to install over the archive

    Mod name Details
    Liberty Tweaks options Improved AI and Remove Weapons on Death have been disabled - you can toggle them back in LibertyTweaks.ini located in IVSDKDotNet\scripts\. You can also tweak your FOV in there, aswell as change keybinds for quicksaving and holstering weapons (F9 and H by default)
    ColAccel by ThirteenAG Speeds up loading times by several times, but can cause memory problems and does not cache moments from the story (like the burnt garage). Can be used in combination with ZolikaPatch's FastLoading option.
    Installation: download version 1.4, extract IV.EFLC.ColAccel.asi into plugins or into the game folder
    DXVK-async 1.10.3 If you have an ancient GPU (like the GTX 600 series) and DXVK from the main archive doesn't work, use this version as a replacement.
    Installation: From the archive, extract d3d9.dll from x32 into the game folder.
    DXVK-async 1.10.1 If you have an Intel iGPU, you can use this version of DXVK.
    Installation: From the archive, extract d3d9.dll from x32 into the game folder.
    Radio Downgrader Brings back removed tracks - which were deleted in 2018 due to licensing issues - to the game.
    Installation: unzip the archive into the folder with the game, open install.bat, then after the console closes itself - extract the update from the desired version of vladivostok (with new vladivostok and without new vladivostok - without will not have new tracks that appeared after the cut).
    Dualshock 4 button textures by tehherb Replaces the Xbox 360 button textures with Dualshock 4 buttons.
    Installation: In modloader edit modloader.ini to change DualshockButtons=1 to 0 at the beginning of the file, and XboxButtons=0 to 1.
    Dualsense button textures by COZlerCae Replaces the Xbox 360 button textures with Dualsense buttons.
    Installation: In modloader edit modloader.ini to change DualsenseButtons=1 to 0 at the beginning of the file, and XboxButtons=0 to 1.
